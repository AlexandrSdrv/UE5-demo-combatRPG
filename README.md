# UE5-demo-combatRPG
Working demo of RPG combat system made with Unreal Engine 5

Features present in the project:
-
- Simple attack with a 3-hit combo if pressing the attack continuously.
- Camera lock-on feature.
- Simple dodging forward by default and 4 sides dodges while the camera is locked on the enemy.
- Self-health bar in the left corner of the screen
- Enemies' health bar above their heads.
- Default enemy AI chasing the player and attacking when reached.
- Boss enemy AI with default attack and a splash attack. 

Description:
-
I did this project for educational purposes to understand how to use the main UE5 components and try to replicate basic RPG combat game features.
